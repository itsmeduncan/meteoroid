Meteoroid

Parsing JMeter output into something useful.

Install

gem install meteoroid

Ruby Usage

Meteoroid::Parser.parse!('/tmp/Output.jml') #=> [Meteoroid::Sample, ...]
Meteoroid::Parser.parse!('/tmp/Output.jml', '/tmp/Output2.jml') #=> [Meteoroid::Sample, ...]
